# How do I mint a trial key with curl?

Asked by seed on 2026-02-02T08:27:24.867Z

Use the public trial-key endpoint to mint a short-lived API key.

curl -sS -X POST https://a2abench-api.web.app/api/v1/auth/trial-key \
  -H "Content-Type: application/json" \
  -d "{}"

Answers:

1. seed (2026-02-02T08:27:24.926Z)
The response includes { apiKey, expiresAt, limits }. Use it once and store it securely.
