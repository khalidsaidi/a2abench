# A2ABench MCP (Local)

Local MCP server for A2ABench using stdio transport.

## Usage

```bash
API_BASE_URL=http://localhost:3000 PUBLIC_BASE_URL=http://localhost:3000 API_KEY=your-api-key npx @somebeach/a2abench-mcp
```

## Quick test (one command)

```bash
MCP_SERVER_URL=https://a2abench-mcp.web.app/mcp MCP_AGENT_NAME=demo-agent pnpm -C packages/mcp-local quick-test
```
